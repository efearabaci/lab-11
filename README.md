## Project Description
A multi-screen React Native app built with Expo that demonstrates:
- Login / Logout flow (mock authentication)
- Global auth state using Context API
- Persisted login session using AsyncStorage
- Persisted theme preference (light / dark) using a custom hook
- Navigation between Login, Home, and Settings screens

## Tech Stack
- React Native (Expo)
- React Navigation (Native Stack)
- AsyncStorage
